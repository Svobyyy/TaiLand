# Tailand — Shopify Pet Shop Theme

Custom Shopify theme for a pet shop brand called **Tailand**.

This project is focused on building a warm, playful, and modern e-commerce experience for pet products.

## Tech / Tools

- Shopify Theme
- Liquid
- CSS
- Shopify CLI
- Custom editable sections
- Responsive design
- Git / GitHub

## Current Features

- Custom brand typography system
- Custom color palette
- Custom header styling
- Custom hero grid section
- Editable Shopify image settings
- Responsive layout
- Portfolio-focused Shopify theme customization

## Design Direction

The design uses a playful pet-shop style with:

- warm cream backgrounds
- orange and green brand accents
- bold typography
- thick bordered cards
- strong offset shadows
- clean e-commerce layout
